# Escola Bíblica IEQ - Sistema PWA

Sistema de gestão para Escola Bíblica desenvolvido como Progressive Web App.

## Funcionalidades
- ✅ Autenticação de usuários
- ✅ Gestão de alunos e professores
- ✅ Registro de frequência com geolocalização
- ✅ Materiais de estudo
- ✅ Bíblia online
- ✅ Relatórios
- ✅ Funciona offline

## Como executar localmente

1. Clone o repositório
2. Abra o arquivo `index.html` em um navegador
3. Para desenvolvimento, use um servidor local (Live Server, Python, etc.)

## Hospedagem no GitHub Pages

1. Faça push para o GitHub
2. Vá em Settings > Pages
3. Selecione branch `main` e pasta `/root`
4. Acesse: https://seusuario.github.io/escola-biblica-pwa/

## Tecnologias
- HTML5, CSS3, JavaScript
- Google Apps Script (backend)
- PWA (Service Workers, Manifest)
- Google Drive API
